# Astro Field Controller

A documented, field-tested setup for running a Windows-based astrophotography rig in a fully headless, battery-powered environment.

This project provides configuration guides, automation scripts, and operational workflows for using a mini PC as a reliable capture and control system with NINA.

---

## Overview

Astro Field Controller is designed for astrophotographers who want a portable, stable, and repeatable way to run their imaging setup without relying on monitors, keyboards, or external networks.

The system focuses on:

- Automatic startup and recovery
- Reliable Wi-Fi hotspot access
- Stable USB and device connections
- Battery-friendly operation
- Minimal user interaction in the field

---

## Key Features

- BIOS and Windows power optimization
- Automatic login and recovery
- Automatic Wi-Fi hotspot startup
- Headless remote operation
- NINA, PHD2, and ASTAP integration
- Field-tested workflows
- Troubleshooting documentation

## Quick Start

1. Review the full setup guide:

   See `ASTRO_FIELD_SETUP.md`

2. Install required software:

   - ASCOM Platform
   - NINA
   - PHD2
   - ASTAP

3. Configure hotspot automation:

   See `scripts/` directory

4. Perform indoor testing before field use.

## Hardware Reference

This setup has been tested on:

- Intel N5105-class fanless mini PC
- Windows 11 Pro
- Portable power stations
- iOptron / ASCOM-compatible mounts
- Canon DSLR cameras

Other similar systems may work with minor adjustments.
